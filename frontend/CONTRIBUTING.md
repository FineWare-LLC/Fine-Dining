## Getting Started

Before you begin, please make sure you have the following installed:

*   Node.js
*   npm

To get the development environment running, follow these steps:

1.  Clone the repository: `git clone https://github.com/your-username/fine-dining.git`
2.  Install the dependencies: `npm install`
3.  Run the development server: `npm run dev`

## Creating New Stories

We aim for 100% line coverage in this application. This means that every line of code you write should be covered by a test.

When creating a new story, please follow these steps:

1.  Create a new branch for your story: `git checkout -b new-story-name`
2.  Write your code and tests.
3.  Make sure your tests cover all of your code.
4.  Run the tests to make sure they all pass: `npm test`

## Testing

We take testing very seriously. In addition to 100% line coverage, we also require that all packages are tested to ensure they stay safe.

Before submitting your changes, please make sure you have done the following:

*   Run all the tests: `npm test`
*   Run a security audit on the packages: `npm audit`

## Code Style

We use Prettier to format our code. Before you commit your changes, please run the following command to format your code:

`npm run format`

## Submitting Changes

Once you have completed your changes, please submit a pull request. We will review your changes and merge them into the main branch as soon as possible.
